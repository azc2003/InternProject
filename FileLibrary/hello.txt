Python is a widely used high-level programming language for general-purpose programming.
Created by Guido van Rossum and first released in 1991, Python has a design philosophy that emphasizes code readability.

Python features a dynamic type system and automatic memory management.
It supports multiple programming paradigms, including object-oriented, imperative, functional programming, and procedural styles.

The language's core philosophy is summarized in the document "PEP 20 (The Zen of Python)".
